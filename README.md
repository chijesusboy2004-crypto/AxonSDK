# AxonSDK Arduino Library

Client SDK for the AXON IoT platform. Allows connecting ESP32 and ESP8266 microcontrollers to your Supabase/Fly.io real-time databases with minimal boilerplate.

## Features

- **Real-Time Telemetry**: Automatically syncs Virtual Pin data to your AXON cloud.
- **Event-Driven Write Callbacks**: Implement Blynk-style `AXON_WRITE(VPIN)` callback macros.
- **4G LTE Cellular Support**: Directly compatible with SIM modules using TinyGSM.
- **ESP-NOW Mesh Support**: Deploy mesh networks using low-power nodes communicating with a central gateway.

## Installation

Search for **AxonSDK** in the Arduino IDE Library Manager and click **Install**.
